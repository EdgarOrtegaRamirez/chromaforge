# Security Policy

## Overview
ChromaForge is a local CLI tool that performs color calculations entirely on your machine. It does not make any network requests, store any data, or access any external resources.

## No Secrets Required
ChromaForge requires no API keys, tokens, or credentials to operate.

## Dependencies
ChromaForge has zero external dependencies — it uses only the Go standard library.

## Reporting
If you discover a security vulnerability, please report it responsibly by opening an issue on GitHub.
